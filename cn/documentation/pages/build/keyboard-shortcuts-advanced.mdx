高级键盘快捷键帮助高级用户在 Dify 中更高效地导航和构建工作流。

## 导航快捷键

### 快速跳转
- `Cmd/Ctrl + K` - 打开命令面板
- `Cmd/Ctrl + P` - 快速文件搜索
- `Cmd/Ctrl + Shift + P` - 打开设置

### 节点选择
- `Tab` - 选择下一个节点
- `Shift + Tab` - 选择上一个节点
- `Cmd/Ctrl + A` - 选择所有节点

## 编辑快捷键

### 节点操作
- `Cmd/Ctrl + D` - 复制选中的节点
- `Cmd/Ctrl + X` - 剪切节点
- `Cmd/Ctrl + C` - 复制节点
- `Cmd/Ctrl + V` - 粘贴节点
- `Delete` 或 `Backspace` - 删除选中的节点

### 画布操作
- `Cmd/Ctrl + Z` - 撤销
- `Cmd/Ctrl + Shift + Z` - 重做
- `Cmd/Ctrl + Plus` - 放大
- `Cmd/Ctrl + Minus` - 缩小
- `Cmd/Ctrl + 0` - 重置缩放至 100%

## 工作流快捷键

### 执行
- `Cmd/Ctrl + Enter` - 运行工作流
- `Cmd/Ctrl + Shift + Enter` - 从选中节点开始运行
- `Esc` - 停止运行工作流

### 测试
- `Cmd/Ctrl + T` - 打开测试面板
- `Cmd/Ctrl + Shift + T` - 运行所有测试

## 提示

<Tip>
  在设置 > 键盘快捷键中自定义键盘快捷键以匹配您的偏好。
</Tip>

<Info>
  大多数快捷键在不同操作系统中都能使用，使用 Cmd (Mac) 或 Ctrl (Windows/Linux)。
</Info>
