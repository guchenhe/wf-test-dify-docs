## 常见任务

### 在工作空间之间切换

1. 点击左上角的工作空间选择器
2. 选择您要切换到的工作空间
3. 或使用 `Cmd+Shift+W` (Mac) / `Ctrl+Shift+W` (Windows/Linux)

### 邀请团队成员

1. 前往 **设置** → **成员**
2. 点击 **邀请成员**
3. 输入电子邮件地址并选择角色
4. 点击 **发送邀请**

### 配置模型提供商

1. 前往 **设置** → **模型提供商**
2. 点击 **添加提供商**
3. 从列表中选择提供商
4. 输入 API 凭据
5. 点击 **保存**

## 工作空间限制

限制因计划而异（仅限云版）：

| 资源 | 免费版 | 专业版 | 团队版 | 企业版 |
|----------|------|-----|------|------------|
| 团队成员 | 1 | 3 | 10 | 无限制 |
| 应用 | 10 | 50 | 200 | 无限制 |
| 知识库 | 5 | 20 | 100 | 无限制 |
| API 调用/月 | 200 | 5,000 | 50,000 | 定制 |

社区版没有限制。

## 故障排除

### 看不到预期的应用

- 检查您的工作空间权限
- 验证您是否在正确的工作空间中
- 如需访问权限，请联系工作空间管理员

### 模型提供商无法工作

- 验证 API 凭据是否正确
- 检查提供商状态页面是否有故障
- 确保有足够的 API 配额/积分

### 未收到邀请邮件

- 检查垃圾邮件/垃圾箱文件夹
- 验证电子邮件地址是否正确
- 管理员可以从成员页面重新发送邀请
