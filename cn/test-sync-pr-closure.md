# 测试同步 PR 关闭

这是一个用于验证 PR 关闭清理工作流的测试文档。

## 目的

此测试验证：
- 当原始 PR 被关闭时，清理工作流触发
- 同步 PR 自动关闭并附带适当的消息
- 合并与关闭 PR 的不同消息
- 重新打开行为（如果适用）

## 测试流程

### 步骤 1：创建 PR
创建此 PR 并包含英文文档更改。

### 步骤 2：等待同步 PR
等待执行工作流创建同步 PR。

### 步骤 3：关闭原始 PR
**不合并**地关闭此 PR 以测试清理工作流。

### 步骤 4：验证清理
验证以下内容：
- 清理工作流检测到关闭
- 同步 PR 自动关闭
- 适当的评论已发布到同步 PR
- 评论指示原始 PR 已关闭（未合并）

### 步骤 5：重新打开（可选）
如果测试重新打开行为：
- 重新打开此 PR
- 验证同步工作流恢复

## 预期清理消息

当未合并关闭时：
> ❌ 原始 PR #XXX 已关闭。如果重新打开，同步将自动恢复。

当合并时：
> ✅ 原始 PR #XXX 已合并。如果翻译已准备好，您仍可以独立合并此同步 PR。
