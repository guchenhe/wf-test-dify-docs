Advanced keyboard shortcuts help power users navigate and build workflows more efficiently in Dify.

## Navigation Shortcuts

### Quick Jump
- `Cmd/Ctrl + K` - Open command palette
- `Cmd/Ctrl + P` - Quick file search
- `Cmd/Ctrl + Shift + P` - Open settings

### Node Selection
- `Tab` - Select next node
- `Shift + Tab` - Select previous node
- `Cmd/Ctrl + A` - Select all nodes

## Editing Shortcuts

### Node Operations
- `Cmd/Ctrl + D` - Duplicate selected node
- `Cmd/Ctrl + X` - Cut node
- `Cmd/Ctrl + C` - Copy node
- `Cmd/Ctrl + V` - Paste node
- `Delete` or `Backspace` - Delete selected node

### Canvas Operations
- `Cmd/Ctrl + Z` - Undo
- `Cmd/Ctrl + Shift + Z` - Redo
- `Cmd/Ctrl + Plus` - Zoom in
- `Cmd/Ctrl + Minus` - Zoom out
- `Cmd/Ctrl + 0` - Reset zoom to 100%

## Workflow Shortcuts

### Execution
- `Cmd/Ctrl + Enter` - Run workflow
- `Cmd/Ctrl + Shift + Enter` - Run from selected node
- `Esc` - Stop running workflow

### Testing
- `Cmd/Ctrl + T` - Open test panel
- `Cmd/Ctrl + Shift + T` - Run all tests

## Tips

<Tip>
  Customize keyboard shortcuts in Settings > Keyboard Shortcuts to match your preferences.
</Tip>

<Info>
  Most shortcuts work across different operating systems with Cmd (Mac) or Ctrl (Windows/Linux).
</Info>
