# Sentiment Analysis Bot

Learn how to create a sentiment analysis bot using Dify workflows.

## Overview

This tutorial covers sentiment analysis implementation using advanced NLP techniques and Dify's powerful workflow capabilities.

## Prerequisites

- Active Dify account
- LLM access (OpenAI, Anthropic, or other providers)
- Basic understanding of sentiment analysis concepts

## Setup

Configure your Dify workspace with the required models and API keys.

### Step 1: Create Workflow

Navigate to your Dify workspace and create a new workflow application.

### Step 2: Add Input Node

Configure the input node to accept text for sentiment analysis.

### Step 3: Configure LLM

Set up an LLM node with a prompt optimized for sentiment detection.

### Step 4: Add Classification

Use a classifier node to categorize results into Positive, Negative, or Neutral.

## Testing

Test your bot with various text samples to ensure accurate sentiment detection.

## Advanced Features

Consider adding:
- Emotion intensity scoring
- Multi-language support
- Custom sentiment categories
