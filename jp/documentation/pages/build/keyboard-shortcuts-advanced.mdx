高度なキーボードショートカットは、パワーユーザーがDifyでより効率的にワークフローをナビゲートし、構築するのに役立ちます。

## ナビゲーションショートカット

### クイックジャンプ
- `Cmd/Ctrl + K` - コマンドパレットを開く
- `Cmd/Ctrl + P` - クイックファイル検索
- `Cmd/Ctrl + Shift + P` - 設定を開く

### ノード選択
- `Tab` - 次のノードを選択
- `Shift + Tab` - 前のノードを選択
- `Cmd/Ctrl + A` - すべてのノードを選択

## 編集ショートカット

### ノード操作
- `Cmd/Ctrl + D` - 選択したノードを複製
- `Cmd/Ctrl + X` - ノードをカット
- `Cmd/Ctrl + C` - ノードをコピー
- `Cmd/Ctrl + V` - ノードをペースト
- `Delete` または `Backspace` - 選択したノードを削除

### キャンバス操作
- `Cmd/Ctrl + Z` - 元に戻す
- `Cmd/Ctrl + Shift + Z` - やり直し
- `Cmd/Ctrl + Plus` - ズームイン
- `Cmd/Ctrl + Minus` - ズームアウト
- `Cmd/Ctrl + 0` - ズームを100%にリセット

## ワークフローショートカット

### 実行
- `Cmd/Ctrl + Enter` - ワークフローを実行
- `Cmd/Ctrl + Shift + Enter` - 選択したノードから実行
- `Esc` - 実行中のワークフローを停止

### テスト
- `Cmd/Ctrl + T` - テストパネルを開く
- `Cmd/Ctrl + Shift + T` - すべてのテストを実行

## ヒント

<Tip>
  設定 > キーボードショートカットで、お好みに合わせてキーボードショートカットをカスタマイズできます。
</Tip>

<Info>
  ほとんどのショートカットは、Cmd（Mac）またはCtrl（Windows/Linux）で異なるオペレーティングシステム間で動作します。
</Info>
