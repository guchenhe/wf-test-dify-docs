Difyは信頼性の高いワークフローを構築するための包括的なデバッグツールスイートを提供しています。

## 組み込みデバッグツール

### デバッグコンソール
デバッグコンソールはリアルタイムの実行情報を表示します：
- ノードの実行ステータス
- 変数の値
- エラーメッセージ
- パフォーマンス指標

### ブレークポイント
特定のノードにブレークポイントを設定して実行を一時停止します：

1. ノードをクリック
2. ブレークポイントのトグルを有効化
3. ワークフローを実行
4. ブレークポイントで実行が一時停止

<Info>
  ブレークポイントはデバッグセッション中のみアクティブで、本番実行には影響しません。
</Info>

## 変数インスペクター

### 複雑なデータの検査
変数インスペクターは以下をサポートします：
- **ネストされたオブジェクト**：オブジェクトのプロパティを展開・折りたたみ
- **配列**：インデックス付きで配列の内容を表示
- **大きな文字列**：切り詰め表示とフルテキストビューアー
- **JSONフォーマット**：シンタックスハイライト付きのJSONビュー

### 変数ウォッチ

```

├─ user_input (string)

```

## ネットワークインスペクター

### API呼び出し監視
ワークフローで行われるすべてのAPI呼び出しを監視：
- リクエストURLとヘッダー
- リクエスト/レスポンスペイロード
- レスポンス時間
- ステータスコード

### API問題のデバッグ
一般的なAPIデバッグシナリオ：
- **タイムアウトエラー**：ネットワーク遅延を確認
- **401/403エラー**：認証資格情報を検証
- **レート制限**：リクエスト頻度を監視

## パフォーマンスプロファイラー

### 実行タイムライン
ワークフロー実行タイムラインを視覚化して以下を特定：
- 長時間実行されるノード
- 順次実行 vs 並列実行
- ノード間の待機時間

### メモリ使用量
メモリ消費を追跡：
- ノードごとのメモリ使用量
- ワークフロー全体のメモリフットプリント
- メモリリーク検出

## ログ集約

### 一元化されたログ
すべてのワークフローログが単一のビューに集約されます：
- 重要度でフィルタリング（Info、Warning、Error）
- キーワードで検索
- 外部分析用にエクスポート可能

<Tip>
  ログ分析を簡単にするために、一貫したフォーマットの構造化ログを使用してください。
</Tip>
