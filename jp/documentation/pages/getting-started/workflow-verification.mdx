## 設定の検証

ワークフローは `tools/translate/config.json` から以下の値を使用する必要があります：

```json
{
  "source_language": "en",
  "target_languages": ["cn", "jp"],
  "languages": {
    "en": {
      "name": "English",
      "directory": "en"
    },
    "cn": {
      "name": "Chinese",
      "directory": "cn",
      "translation_notice": "<Note>...</Note>"
    },
    "jp": {
      "name": "Japanese",
      "directory": "jp",
      "translation_notice": "<Note>...</Note>"
    }
  }
}
```
