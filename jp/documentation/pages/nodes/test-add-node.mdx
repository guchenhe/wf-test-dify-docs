テスト追加ノードは以下を実証します：

- 英語から中国語と日本語への新規ファイル翻訳

- インデックスベースの配置を使用したナビゲーション構造内の正しい配置

## テスト検証

このファイルがリポジトリに追加されると：

1. 翻訳ワークフローは英語セクション内のこの新しいファイルを検出する必要があります

3. 両方のターゲット言語のdocs.jsonに同じインデックス位置でエントリを挿入します
4. 翻訳されたファイルの冒頭に翻訳通知を追加します
