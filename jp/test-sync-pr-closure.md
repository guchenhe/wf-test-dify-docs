# テスト同期PRクローズ

これはPRクローズクリーンアップワークフローを検証するためのテストドキュメントです。

## 目的

このテストは以下を検証します：
- 元のPRがクローズされたときにクリーンアップワークフローがトリガーされること
- 同期PRが適切なメッセージとともに自動的にクローズされること
- マージされた場合とクローズされた場合で異なるメッセージが表示されること
- 再オープン動作（該当する場合）

## テスト手順

### ステップ1: PRを作成
英語ドキュメントの変更を含むこのPRを作成します。

### ステップ2: 同期PRを待つ
Executeワークフローが同期PRを作成するのを待ちます。

### ステップ3: 元のPRをクローズ
クリーンアップワークフローをテストするために、このPRを**マージせずに**クローズします。

### ステップ4: クリーンアップを確認
以下を確認します：
- クリーンアップワークフローがクローズを検出すること
- 同期PRが自動的にクローズされること
- 適切なコメントが同期PRに投稿されること
- コメントが元のPRがクローズされた（マージされていない）ことを示していること

### ステップ5: 再オープン（オプション）
再オープン動作をテストする場合：
- このPRを再オープンします
- 同期ワークフローが再開されることを確認します

## 期待されるクリーンアップメッセージ

マージせずにクローズした場合：
> ❌ Original PR #XXX was closed. If it reopens, sync will resume automatically.

マージした場合：
> ✅ Original PR #XXX was merged. You can still merge this sync PR independently if translations are ready.
