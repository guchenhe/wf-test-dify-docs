## 設定の詳細

新しい統合設定により、`config.json` を編集するだけで新しい言語を追加できます：

```json
{
  "source_language": "en",
  "target_languages": ["cn", "jp"],
  "languages": {
    "en": {
      "code": "en",
      "name": "English",
      "directory": "en"
    },
    "cn": {
      "code": "cn",
      "name": "Chinese",
      "directory": "cn",
      "translation_notice": "..."
    }
  }
}
```

新しい言語を追加するためにコード変更は不要です！
